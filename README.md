# MyFinance - Hệ Thống Quản Lý Khoản Vay Tiêu Dùng

Dự án website quản lý khoản vay cá nhân với giao diện hiện đại, tích hợp biểu đồ thống kê và hệ thống thông báo chuyên nghiệp.

## 🚀 Tính năng chính

- **Dành cho Khách hàng:**
  - Đăng ký khoản vay linh hoạt.
  - Xem thống kê dư nợ bằng biểu đồ trực quan.
  - Theo dõi lịch trả nợ chi tiết từng tháng.
  - Thanh toán trả góp trực tuyến.
- **Dành cho Admin:**
  - Phê duyệt hoặc từ chối khoản vay kèm ghi chú.
  - Quản lý danh sách khách hàng và dư nợ toàn hệ thống.
  - Dashboard thống kê tình hình kinh doanh bằng biểu đồ.
- **Tính năng nổi bật:**
  - Giao diện Glassmorphism hiện đại, hỗ trợ Chế độ tối (Dark Mode).
  - Hệ thống thông báo Toast cao cấp thay cho alert mặc định.
  - Hiệu ứng chuyển động mượt mà, trải nghiệm người dùng tối ưu.

## 🛠 Công nghệ sử dụng

- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+), FontAwesome, Chart.js.
- **Backend:** Node.js, Express.
- **Database:** SQL Server (MSSQL).

## 📦 Hướng dẫn cài đặt

1.  **Clone dự án:**
    ```bash
    git clone https://github.com/phamchithanh0905/VayTieuDung.git
    cd VayTieuDung
    ```
2.  **Cài đặt dependencies:**
    ```bash
    npm install
    ```
3.  **Khởi tạo Database:**
    - Chạy script trong file `init.sql` trên SQL Server Management Studio (SSMS).
4.  **Chạy server:**
    - Cách 1: Chạy file `Run-Server.bat`.
    - Cách 2: `node server.js`.
